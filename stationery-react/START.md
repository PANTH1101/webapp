# 🚀 Quick Start Guide

## Start the Development Server

```bash
npm run dev
```

Then open your browser to: **http://localhost:5173**

## Build for Production

```bash
npm run build
```

The production files will be in the `dist/` folder.

## Preview Production Build

```bash
npm run preview
```
