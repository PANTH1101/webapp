## 🚀 Quick Start

### 1. Start Development Server

```bash
cd stationery-react
npm run dev
```

Open: **http://localhost:5173**

### 2. Build for Production

```bash
npm run build
```

Output: `dist/` folder

### 3. Preview Production Build

```bash
npm run preview
```
